Typography

GrooveGraph uses Plus Jakarta Sans as its primary typeface — a geometric sans-serif with excellent legibility at small sizes and distinctive character at display weights. Monospace code uses JetBrains Mono.

Colorways

The base palette uses HSL-based CSS custom properties with light and dark variants. Graph nodes and edges use a separate, high-saturation palette tuned for on-card contrast. Toggle dark mode with the moon button in the header.

Graphics & Brand

The GrooveGraph wordmark uses a gradient across the primary entity colors — Artist purple to Track blue to Album green. The logo mark is a stylized graph node cluster representing connected music entities.
